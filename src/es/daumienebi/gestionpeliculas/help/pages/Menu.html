Menú

The top menu bar consists of the following items
  • Connection: Option to set the database connection and other required config values.Connection settings
  • Home: Shows the options to exit the app or refresh the data tables
  • Actors: The Actors management options.Actor Management
  • Movies: The Movies management options.Movie Management
  • Genres: The menu option only shows the available movie genres
  • Reports: Menu Option to access the available incrusted reports. Check out the Report help
  • Settings: The available configurable options for the app [switching languages between English and Spanish]
